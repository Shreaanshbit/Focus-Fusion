# FocusFusion Tabs & Graphs - A productivity tool for managing tasks and visualizing progress through interactive tabs and graphs

FocusFusion is a modern desktop application that helps you boost productivity and maintain focus by combining task management with visual progress tracking.

## Features

- **Smart Tab Management**: Organize your work into intuitive tabs for better context switching
- **Visual Progress Tracking**: Interactive graphs to visualize your productivity patterns
- **Clean Interface**: Minimalist PyQt5-based UI designed for distraction-free work
- **Real-time Updates**: Dynamic graphs that update as you complete tasks
- **Customizable Workspace**: Arrange and organize tabs according to your workflow

## Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/FocusFusion_Tabs_Graphs.git
cd FocusFusion_Tabs_Graphs
```

2. Create and activate a virtual environment (optional but recommended):
```bash
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
```

3. Install dependencies:
```bash
pip install -r requirements.txt
```

## Usage

1. Launch the application:
```bash
python main.py
```

2. Create new tabs for different projects or contexts
3. Add tasks and track your progress
4. View your productivity patterns in the graphs section
5. Use the insights to optimize your workflow
