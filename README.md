# FocusFusion Tabs & Graphs - A productivity tool for managing tasks and visualizing progress through interactive tabs and graphs

A PyQt5-based application for managing tabs and displaying graphs.

## Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/FocusFusion_Tabs_Graphs.git
cd FocusFusion_Tabs_Graphs
```

2. Create and activate a virtual environment (optional but recommended):
```bash
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
```

3. Install dependencies:
```bash
pip install -r requirements.txt
```

## Usage

Run the application:
```bash
python main.py
```
